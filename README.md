Blog
====

Simple java blog with gplus login.

Stack
=====
 - mongodb & com.github.jmkgreen.morphia
 - jersey
 - com.google.apis
 - (a little) freemarker
 - angularjs

run
=====
1. Try

        mvn compile package && java -jar target/jlog-0.0.1-SNAPSHOT.jar
2. See it fail
3. Add proper `nu.wasis.blog.util.PrivateConstants.java`
4. ...
5. Profit
