jlog
====

Simple java blog with gplus login.

Stack
=====
 - mongodb & com.github.jmkgreen.morphia
 - jersey
 - com.google.apis
 - (a little) freemarker
 - angularjs
 - bootstrap
 - (tiny portions of) jquery
 - flippant.js
